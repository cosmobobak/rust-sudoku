# rust-sudoku
Sudoku solver in Rust

This solver has support for both 9x9 and 16x16 sudokus, and typically runs in <2ms for a 9x9.

Example usage:

$ cargo run -q --release -- --9------384---5------4-3-----1--27-2--3-4--5-48--6-----6-1------7---629-----5---
